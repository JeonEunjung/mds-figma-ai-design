# Colors

Figma 프레임/텍스트를 수동으로 만들 때 사용하는 색상 토큰.
컴포넌트 인스턴스는 자체 색상이 적용되어 있으므로 이 값을 쓸 필요 없음.

## Surface (배경색)
| 용도 | hex |
|------|-----|
| 페이지 배경 | #FFFFFF |
| 카드/보조 배경 | #F9FAFB |
| 강조 배경 파랑 | #0066FF |
| 에러 배경 | #FEF2F2 |
| 성공 배경 | #F0FDF4 |
| 경고 배경 | #FFFBEB |
